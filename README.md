# xia SQL (瞎注)

> 本插件仅只插入单引号，没有其他盲注啥的，而且结果返回响应包的长度需要人工去判断是否存在注入，如果需要所有注入都测试，请把burp的流量转发到xray。

* burp 插件。
* 在每个参数后面填加一个单引号，两个单引号。
* 由于不会java，且又是用java写的，代码太烂，就不开源了。
